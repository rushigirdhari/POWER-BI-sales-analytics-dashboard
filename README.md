# Power BI Sales Analytics Dashboard

## Project Overview
This project presents an interactive Sales Analytics Dashboard built using Power BI. The dashboard provides insights into sales performance, customer behavior, and revenue trends using interactive visualizations.

## Features
- KPI Cards for Revenue, Profit, and Orders
- Sales Trend Analysis
- Customer Segmentation
- Geographic Sales Distribution
- Interactive Filters and Slicers

## Tools Used
- Power BI
- Power Query
- DAX

## Insights
The dashboard helps analyze sales patterns, identify top customers, and understand revenue growth over time.
